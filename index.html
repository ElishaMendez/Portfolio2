Pharmacy Capacity Model Validation & Optimization

Analyzed theoretical capacity model for a 10-person pharmacy team to identify opportunities for improving daily output and operational efficiency.

Planned Features

  • Extracted and aggregated production data across 18 time slots (8:30 AM - 5:00 PM) to calculate total output metrics
  • Validated theoretical vs. actual output projections (3,957.5 orders/day target) using Excel formulas
  • Analyzed performance variance across staff using comparative metrics (600 vs 300 orders/shift range)
  • Applied time-series analysis to evaluate hourly productivity patterns (1,874 orders by 11 AM vs 652 orders 11 AM-1 PM)
  • Assessed task distribution workflows (Fill, Prefill, Packing, Shipping) for optimization opportunities
  • Developed data tracking framework using Excel to monitor model accuracy and identify performance gaps
  • Created actionable recommendations prioritized by implementation complexity and data requirements

Tools: Excel
