## Project: Design a store database - For this project, using SQL I created a Pharmacy Inventory Database with Summary Statistic. The goal was to practice creating tables, inserting real-world medication data, and running queries that organize and summarize inventory information. This helped me apply what I learned in a practical way and build a basic system that could track stock levels, pricing, and product availability.
